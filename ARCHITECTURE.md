Deployment Architecture

- Public data sources (news, tenders, websites)
- Ingestion service triggers inference API
- Inference engine processes signals and generates lead dossiers
- Leads stored and routed to sales officers
- Mobile web app for lead consumption
- Notifications via WhatsApp / email (sandbox)

Architecture is event-driven and horizontally scalable.
Estimated cost: ₹30–50 per lead at scale.
